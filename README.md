Text Lowercase Converter

This is a simple Python script that converts any text input by the user into lowercase letters.

How It Works

The script prompts the user to enter some text.

It converts the input text to all lowercase letters.

It prints the lowercase version of the text.

Example Usage
Input:  Hello World!
Output: hello world!

How to Run

Make sure Python is installed on your system.

Run the script in a terminal or command prompt:

python filename.py


Enter your text when prompted and see the output in lowercase.
